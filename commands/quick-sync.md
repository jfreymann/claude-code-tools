# Quick Sync

Lightweight sync that only updates the changelog. Use for quick context preservation without full documentation review.

## Process

1. **Summarize session** briefly:
   ```markdown
   ### [Date] - [Brief Title]

   **Focus**: [Main topic]

   **Done**:
   - [2-4 key items]

   **Notes**:
   - [Important context]

   **Next**:
   - [ ] [Immediate follow-ups]
   ```

2. **Prepend to docs/changelog.md**

3. **Update claude.md timestamp**

4. **Output confirmation**

## When to Use

- End of short sessions
- Quick checkpoint during long work
- When /sync feels like overkill

## Start

Summarize this session for the changelog.
